# 记忆增强功能

## 功能说明

为新闻推送添加历史上下文，让每次推送更有深度和洞察力。

## 增强内容

### 1. 历史关联
```
DeepSeek 发布 v4-flash 模型
💡 关联记忆：2周前 DeepSeek V3 发布
```

### 2. 趋势提醒
```
DeepSeek 发布 v4-flash 模型  
📈 趋势提醒：话题热度暴涨 275%
```

### 3. 话题洞察
```
持续话题：AI 基础设施、智能汽车
突发事件：DeepSeek 新品发布
冷却话题：区块链、元宇宙
```

### 4. 知识库统计
```
累积记忆：15 条
本周新增：7 条
最新更新：2026-04-26
```

## 配置方法

在 `config/config.yaml` 中添加：

```yaml
MEMORY:
  # 启用推送增强
  ENHANCE_NOTIFICATION: true
```

## 工作原理

1. **新闻推送前**：从记忆数据库查询相关历史
2. **提取关键词**：识别新闻标题中的重点话题
3. **关联记忆**：查找相关的每日摘要和周汇总
4. **趋势分析**：计算关键词热度变化
5. **格式化输出**：在原始推送内容后添加增强信息

## 技术实现

- **数据源**：memory.db + news/*.db
- **查询方式**：关键词匹配 + 时间范围过滤
- **性能影响**：每次推送增加约 50-100ms

## 注意事项

1. 需要先运行几天积累记忆数据
2. 首次使用可能没有历史关联
3. 关键词识别基于预定义列表，可能遗漏某些话题
4. 如果推送速度明显变慢，可以禁用此功能
